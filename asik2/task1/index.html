Welcome

This page is an example of a simple but stylish two-column layout. It shows how colors, borders, and spacing can make a clean design.

On the left side you can see a sidebar with quick links. On the right side there is the main content. The goal is to create a structure that looks modern but is still very easy to understand.

Highlights
  • Clean header with gradient background and subtle shapes.
  • Navigation bar with hover effects and active state.
  • Two-column layout built using floats only.
  • Responsive design: columns stack on smaller screens.
